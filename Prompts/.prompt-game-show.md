# Retro Game Show Scoreboard - "Schlag den Köln" Style

## Purpose
Generate a retro-styled game show interface inspired by German TV game shows (specifically "Schlag den Köln"). The app displays a live scoreboard for 2 players competing in a 15-game tournament, with a spinner wheel for game selection and point tracking.

## Visual Style & Design
**Aesthetic**: Authentic 2000s game show style
- Primary color: Bright **yellow background** (#FFD700, #FFC107)
- Accent colors: Black for text/borders, white for highlights
- Bold, chunky fonts (Arial Black, Impact, similar)
- Large, easy-to-read numbers and text
- Decorative elements: Gold/silver 3D effects, glowing borders, sunburst patterns
- Reference: "Schlag den Köln" TV show visual theme

**Key UI Components**:

### 1. Scoreboard (Top Section)
- Player 1 & Player 2 names in large bold text
- Current score display (large, prominent numbers)
- Small box showing games already played (1, 2, 3, with visual indicators)
- Games remaining to play shown as empty boxes
- Overall tournament progress visually indicated

### 2. Spinner Wheel (Center/Right Section)
- Large spinning wheel graphic
- Numbers 1-15 arranged around the wheel circumference
- Visual segments in alternating colors (yellow/green or yellow/blue)
- "NACH X SPIELEN" label showing next milestone
- Spin button to randomize game selection
- Current selected game highlighted prominently

### 3. Points Display (Right Side)
- "GESPIELTE PUNKTE" (Points Already Earned) - large black box with white text
- "OFFENE PUNKTE" (Points Still Available) - large black box with white text
- Updated in real-time as games complete
- Shows both players' tallies

### 4. Game Selection & Control
- "Weiter" (Continue) button to proceed after spinning
- Game number/status updates
- Ability to mark game as completed with winner selection

## Technical Specifications

### Output Format
- Single HTML file with embedded CSS and JavaScript
- No external frameworks required (vanilla JavaScript)
- CDN for Chart.js only (for optional advanced charts)
- Fully responsive but optimized for desktop/large screens
- Ready to open in browser immediately

### Game Flow
1. Player spins the wheel to select next game (1-15)
2. Players play the game externally (app is coordinator only)
3. User indicates which player won the game
4. Points awarded automatically
5. Scoreboard and remaining points update
6. Visual feedback (animations, color changes) on score updates
7. Progress tracked throughout tournament

### Data Tracking
- Player names (input or defaults: "Player 1", "Player 2")
- Current score for each player
- Completed games list
- Remaining games list
- Points available to win
- Games played/remaining progress
